# Share of population living in extreme poverty - Data package

This data package contains the data that powers the chart ["Share of population living in extreme poverty"](https://ourworldindata.org/explorers/poverty-explorer?Indicator=Share+in+poverty&Poverty+line=%243+per+day%3A+International+Poverty+Line&Household+survey+data+type=Show+data+from+both+income+and+consumption+surveys&Show+breaks+between+less+comparable+surveys=false&country=BGD~BOL~KEN~MOZ~NGA~ZMB) on the Our World in Data website.

## Column Descriptions

| Column | Type | Description |
|--------|------|-------------|
| `country` | Categorical | Country name (ISO standard) |
| `year` | Integer | Survey year of data collection |
| `headcount_ratio_300` | Numeric (0-100) | **Poverty headcount ratio at $3.00/day (2017 PPP)**<br>**Percentage of population** living below **$3.00 per day**<br>• Lower-middle income country poverty line<br>• Measures **extreme poverty**<br>• Household surveys (income/consumption per capita)<br>• After taxes and benefits |
